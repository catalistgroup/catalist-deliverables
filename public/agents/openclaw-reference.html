Common Fix Patterns
OAuth Token Expired
Evidence: 401 errors, multiple agents failing
refresh-token.sh → wait 15s → ops.sh restart
Session Bloat
Evidence: .jsonl files > 2MB, slow responses
session-pruner-v2.sh → ops.sh restart
chattr +a Lock (EPERM)
Evidence: "Operation not permitted" in logs
chattr-strip-sessions.sh
Gateway Down
Evidence: all agents unresponsive
ops.sh restart
Cron Timing Overlap
Evidence: agent timeout, queue congestion
Edit cron/jobs.json, stagger by 10+ min → restart
Anthropic API Outage
Evidence: 5xx/529 errors, all agents failing
Wait for recovery → reset error counts → restart
